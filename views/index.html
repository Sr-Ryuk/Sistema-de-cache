API Node.js

Documentação da API RESTful

Endpoints Disponíveis

Página Padrão

GET /

Retorna informações básicas sobre a API.

Clientes

Listar todos os clientes

GET /clientes

Retorna uma lista de todos os clientes.

Filtros opcionais: ?nome=João&email=joao@exemplo.com&idade=30

Obter cliente por ID

GET /clientes/:id

Retorna um cliente específico pelo ID.

Criar novo cliente

POST /clientes

Cria um novo cliente.

Corpo da requisição:

{
  "nome": "João",
  "sobrenome": "Silva",
  "email": "joao.silva@exemplo.com",
  "idade": 30
}

Atualizar cliente

PUT /clientes/:id

Atualiza um cliente existente.

Corpo da requisição (todos os campos são opcionais):

{
  "nome": "João",
  "sobrenome": "Silva",
  "email": "joao.silva@exemplo.com",
  "idade": 31
}

Excluir cliente

DELETE /clientes/:id

Remove um cliente pelo ID.

Produtos

Listar todos os produtos

GET /produtos

Retorna uma lista de todos os produtos.

Filtros opcionais: ?nome=Smartphone&preco_min=100&preco_max=2000

Obter produto por ID

GET /produtos/:id

Retorna um produto específico pelo ID.

Criar novo produto

POST /produtos

Cria um novo produto.

Corpo da requisição:

{
  "nome": "Smartphone",
  "descricao": "Smartphone com 128GB de memória",
  "preco": 1299.99
}

Atualizar produto

PUT /produtos/:id

Atualiza um produto existente.

Corpo da requisição (todos os campos são opcionais):

{
  "nome": "Smartphone Pro",
  "descricao": "Smartphone com 256GB de memória",
  "preco": 1499.99
}

Excluir produto

DELETE /produtos/:id

Remove um produto pelo ID.
